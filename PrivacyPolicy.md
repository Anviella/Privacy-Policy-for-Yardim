# Privacy Policy for Yardim#6615 
This Privacy Policy explains how the Yardim#6615's collects, uses, and protects user data.

# 1. Data Collected

The Ticket Bot only collects the minimum data necessary to perform its functions. The collected data includes:

Server ID
Support Category ID 

Ticket (support request) numbers 

Ticket (support request) content 

Ticket transcripts 

This data is collected solely to ensure the proper functioning of the bot and to manage the support system.

# 2. Purpose of Data Usage

The collected data is used only for the following purposes:

To operate the ticket (support) system
To manage and respond to user support requests
To generate and review support transcripts when necessary

The Ticket Bot does not use this data for any other purposes.

# 3. Data Sharing

Collected data is not shared, sold, or rented to any third parties.

# 4. Data Retention

Data is retained only for as long as necessary. Ticket content and transcripts are stored as part of the support process and may be deleted when no longer needed.

# 5. Data Security

The Ticket Bot implements reasonable security measures to protect collected data. However, no method of data transmission over the internet can be guaranteed to be completely secure.

# 6. Purpose of the Bot

The sole purpose of the Yardim#6615 is to:

Provide a support (ticket) system
Provide a server registration system

It does not have any other functionality or purpose beyond these.

# 7. User Rights

Server administrators can stop data collection at any time by removing the bot from their servers.

# 8. Changes

This Privacy Policy may be updated from time to time. Any changes will become effective upon being published.

# 9. Contact

If you have any questions regarding this Privacy Policy, you may contact the bot developer. #
